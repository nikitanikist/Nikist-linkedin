# Strategy Planner — LinkedIn Growth Engine

Create comprehensive LinkedIn growth plans with clear milestones, daily rhythms, and scaling roadmaps. This is the orchestrator that ties all other skills together.

## Core Philosophy

From the course: "Speed over perfection. Something is better than nothing. The people who win on LinkedIn are the ones who start before they're ready and iterate based on data."

## Step 1: Assess Current State

Before creating a plan, understand where the user is. Ask via AskUserQuestion:

1. **LinkedIn profile status**: New / Exists but not optimized / Actively posting
2. **Current metrics**: Followers, weekly impressions, connection requests/week, meetings/month
3. **Business stage**: Pre-revenue / Early revenue / Established / Scaling
4. **Time available**: How many hours/day can they dedicate to LinkedIn?
5. **Goal**: Build personal brand / Generate leads / Both
6. **Timeline**: 30 days / 90 days / 6 months

## Step 2: The 30/60/90 Day Roadmap

### Days 1-30: Foundation (Execute, Don't Perfect)

**Week 1 — Setup:**
- Run ICP Builder → Define target audience
- Run Profile Optimizer → Optimize all profile elements
- Identify 10 creators for engagement (Engagement Bot)
- Set up content calendar (Content Engine)

**Week 2 — Content Launch:**
- Post first 3-4 pieces of content
- Start daily engagement routine (30 min/day)
- Begin profile viewing (20-30 ICP profiles/day)
- Track: impressions, profile views, connection requests

**Week 3 — Outreach Begins:**
- Start sending 20-25 connection requests/day
- Send first messages to new connections
- Create first lead magnet
- Track: messages sent, reply rate, connections accepted

**Week 4 — Optimize:**
- Review metrics from Weeks 1-3
- Double down on content types that performed best
- Adjust outreach messaging based on reply rates
- Book first discovery calls
- Create personal playbook documenting what's working

### Days 31-60: Scale

**Week 5-6:**
- Increase posting to 4-5x/week
- Launch 2nd lead magnet
- Refine outreach messaging (test 4 variations)
- Start tracking conversion funnel metrics
- Target: 5-10 meetings/month

**Week 7-8:**
- Introduce content batching (create 2 weeks at once)
- Expand outreach to 30+ messages/day
- Start warm outreach to profile viewers
- Explore automation tools (Heritage, Make)
- Target: 10-15 meetings/month

### Days 61-90: Compound Effects

**Week 9-10:**
- Content should be generating inbound leads
- Outreach should have 2-3 message templates that convert
- Lead magnets should be your #1 meeting generator
- Target: 15-25 meetings/month

**Week 11-12:**
- Build playbook for delegation (if scaling to team)
- Implement automation where proven (not before testing manually)
- Start planning pricing tiers for services
- Target: 20-30 meetings/month consistently

## Daily Operating Rhythm

The ideal LinkedIn day (60-90 minutes total):

**Morning (30 min) — Engagement:**
- 10-Creator engagement routine (Engagement Bot)
- Respond to comments on your posts
- Reply to DMs

**Midday (15 min) — Content:**
- Post scheduled content
- Check performance of yesterday's post
- Engage with comments on your new post

**Afternoon (30-45 min) — Outreach:**
- Send 20-25 connection requests
- Send first messages to new connections
- Follow up with non-responders
- Book discovery calls

**Weekly Review (30 min, Sunday/Monday):**
- Review all metrics
- Plan next week's content
- Adjust strategy based on data

## Scaling Framework: Freelancer → Agency

From the course, the 6-month progression:

**Month 1-2: Solo Execution**
- Do everything yourself
- Document what works in a personal playbook
- Close 2-3 deals to prove the model

**Month 3-4: Systematize**
- Build playbook (80-130 pages covering all processes)
- Create templates for everything
- Price goes up (you now have proof)

**Month 5-6: Delegate**
- Hire first team member (VA or junior)
- Train using playbook
- Focus on strategy and closing, delegate execution
- Target: ₹5-10L/month or $5-10K/month

## GTM Strategy via LinkedIn

For SaaS founders specifically:

1. **Position as founder, not vendor** — share building journey
2. **Content = top of funnel** — educate your ICP
3. **Lead magnets = middle of funnel** — capture interest
4. **DMs = bottom of funnel** — book demos
5. **Discovery calls = conversion** — close deals

Channel strategy by deal size:
- Under $500 ACV: Content + inbound only
- $500-$5,000 ACV: Content + targeted outreach
- $5,000+ ACV: Full outreach machine + content + engagement

## Anti-Patterns

- Don't skip the foundation (profile + ICP) to jump to outreach
- Don't automate before testing manually
- Don't change strategy every week — give it 30 days
- Don't focus on follower count — focus on meetings booked
- Don't compare Day 1 to someone's Year 3
- Don't work without metrics — what you don't measure, you can't improve
- Don't ignore the compound effect — Month 3 is exponentially better than Month 1
